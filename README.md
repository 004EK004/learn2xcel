## Project Summary
Learn2Excel is a comprehensive bootcamp website utilizing Next.js 16, TypeScript, and Tailwind CSS v4. This website integrates Appwrite for authentication and database management, providing users with a polished experience in AI-driven Excel courses, original Excel content, and data analysis.

## Key Highlights
- **Protected Dashboard**: Secure access to user-specific content.
- **Bootcamp Enrollment**: Users can easily enroll in bootcamps.
- **Contact & Waitlist Forms**: Streamlined communication and user management.
- **SEO Routes**: Optimized for better search engine visibility.
- **Customized Theme**: Tailored UI components for a unique user experience.

## Tech Stack
- **Frontend**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Database**: Appwrite

## Features
- User Authentication via Appwrite
- Courses on AI with Excel, Data Analysis, etc.
- Responsive Design
- User Notifications through Toast Messages

## Routes
- `/` : Home Page
- `/dashboard` : User Dashboard (protected)
- `/enroll` : Enrollment Form
- `/contact` : Contact Form
