## Project Summary
Learn2Excel is a comprehensive bootcamp website utilizing Next.js 16, TypeScript, and Tailwind CSS v4. The experience delivers polished, conversion-focused pages for AI-driven Excel courses, original Excel content, and data analysis tracks.

## Key Highlights
- **Protected Dashboard**: Secure access to user-specific content.
- **Bootcamp Enrollment**: Users can easily enroll in bootcamps.
- **Contact & Waitlist Forms**: Streamlined communication and user management.
- **SEO Routes**: Optimized for better search engine visibility.
- **Customized Theme**: Tailored UI components for a unique user experience.

## Tech Stack
- **Frontend**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Data/Auth Layer**: Managed backend service (configured via environment variables)

## Features
- Secure user authentication
- Courses on AI with Excel, Data Analysis, etc.
- Responsive Design
- User Notifications through Toast Messages

## Design System (Premium Refresh)
- **Visual language**: Emerald-led gradient palette with premium white card surfaces, soft borders, and elevated shadow tokens.
- **Component style**: Rounded-xl/3xl cards, glassmorphic highlights, and consistent CTA styling for conversion-focused flows.
- **Motion system**: Subtle hover lift/scale, staggered fade-ins, and lightweight mouse-reactive gradients for key banners.
- **Typography & spacing**: Stronger heading hierarchy, improved line-height, and cleaner section rhythm across desktop and mobile.
- **Trust signals**: Guarantee messaging and outcome-focused benefit bullets integrated into conversion sections.

## Routes
- `/` : Home Page
- `/dashboard` : User Dashboard (protected)
- `/enroll` : Enrollment Form
- `/contact` : Contact Form
